# 银杏 Launcher UI/UX 优化提案 — 高保真原型

更新时间：2026-04-26
状态：设计审阅稿（未开发）

## 这是什么

这是为银杏 Launcher 准备的 **UI / 交互 / 信息架构** 整套优化提案的可视化原型，所有内容以网页形式呈现，可在浏览器中直接打开 `index.html` 浏览。

## 为什么要做这个

当前 Android 工程已经有相对成体系的设计系统（`@/d:/OldLauncher/app/src/main/res/values/typography.xml`），但在以下方面对老人体验仍有明确硬伤：

- 暗色模式形同虚设（`values-night/themes.xml` 是空注释）
- 联系人卡片只有按钮没有号码，按错直接拨出
- 删除/危险操作与取消按钮等宽，没有二次确认
- 设置页 6 张卡片全展开，要滚动 2-3 屏
- "长按删除应用"作为隐藏手势，老人发现不了
- 设置页有大量绕过 `typography` 的硬编码字号
- 顶部 Hero 卡片在 4 个页面重复，padding 各不相同

## 原型范围

| 模块 | 现状（左） | 优化后（右） |
|------|-----------|-------------|
| 主页 | 时钟/天气/2x N 应用网格 | 同上 + 农历/节日提示 + 维护中心入口收拢 |
| 电话联系人 | 头像+姓名+按钮 | 头像+姓名+**号码**+按住拨号 |
| 视频联系人 | 同电话 | 同电话 |
| 设置 | 6 卡片瀑布流 | 分组 + 状态优先级 |
| 拨号确认 | 直接拨出 | 按住 1 秒确认 |
| 删除确认 | 等宽按钮 | 危险操作收窄 + 长按确认 |
| 暗色模式 | 不存在 | 完整暗色体系 |

## 怎么看

- 完整的 **设计 token 表**（颜色、字号、圆角、阴影）
- 与当前代码的 **diff 对照表**
