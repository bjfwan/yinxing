# 当前功能状态

更新时间：2026-04-15

| 功能 | 状态 | 当前说明 |
| --- | --- | --- |
| 桌面主页 | 已实现 | 支持时间日期展示、系统设置、电话、视频页、天气和已选择应用入口 |
| 默认 Launcher 入口 | 已实现 | Manifest 已声明 `HOME` 和 `LAUNCHER` |
| 应用管理 | 已实现 | 支持勾选加入桌面和取消移除，基于 `LauncherAppRepository` 读取并缓存桌面应用 |
| 天气入口 | 已实现 | 优先尝试厂商天气应用，失败时回退到浏览器天气页 |
| 电话联系人 | 已实现 | 支持读取、添加、编辑、删除和拨号，无通话权限时回退到系统拨号页，并包含空状态、权限拒绝和失败提示 |
| 视频联系人管理 | 已实现 | 页面可独立打开，支持本地新增、删除、搜索、置顶和按通话次数排序 |
| 微信视频自动化拨号 | 已验证 | 在 vivo V2285A / Android 15 / 微信 8.0.66 上验证通过；无障碍服务类名采用 `com.google.android.accessibility.selecttospeak.SelectToSpeakService` 触发微信节点树开放 |
| 微信视频自动接听 | 未实现 | 没有通知监听服务和来电处理链路 |
| 系统电话自动接听 | 未实现 | 没有电话状态监听和接听流程 |
| 自动化状态浮窗 | 已实现 | 服务于微信自动化链路的浮窗状态播报 |
| 自动化 TTS 播报 | 已实现 | 服务于微信自动化链路的语音播报 |
| 自动化测试体系 | 部分实现 | 已建立 `testDebugUnitTest`、`assembleDebug`、`assembleDebugAndroidTest`、`lintDebug` 验收动作，补齐主干页面 Robolectric UI 冒烟，并新增第一批主页、设置、视频联系人页设备级仪器测试脚手架 |

## 说明

- "已实现"表示主流程在当前代码中已经存在
- "已验证"表示主流程在真实设备上跑通，但兼容性覆盖尚不完整
- "部分实现"表示存在入口，但仍有明显限制或兼容性缺口
- "未实现"表示需求或规划存在，但当前仓库没有完整交付链路
- 视频联系人页不再依赖实验性自动化权限才能打开，相关权限在真正发起微信视频时单独检查
- 微信自动化无障碍服务（`SelectToSpeakService`）已合并为单一服务，用户在设置中只需开启一个无障碍项即可
