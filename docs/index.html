这套颜色不是为设计师做的，是按老花眼、对比敏感度下降、晶状体变黄等真实视觉变化反推出来的。所有改动基于 colors.xml 增量扩展，不破坏 110+ 处引用。

01 为老人 · 红绿灯逻辑

绿 / 黄 / 红 = 走过几十年的颜色

老人在马路上看了一辈子红绿灯，不需要重新学习。所以绿色只给安全的操作（接听、保存），黄色只给需要确认（倒计时），红色只给不可逆（挂断、删除）。考虑到老年人红绿色弱发生率上升，每个动作还配文字 + 图标双重提示，不靠颜色单独表意。

02 为老人 · 暖色感知偏好

银杏黄 取自他们熟悉的树叶

老年人晶状体逐年发黄，蓝光透过率下降，暖色辨识反而变好。所以品牌色没用流行的冷蓝紫，而是取#B8882A 银杏叶颜色——温暖、亲切、不刺激；柔淡变体 #FBF1DC 做大块背景，文字对比度仍稳在 WCAG AAA 级（≥7:1），老花眼也看得清。

03 为老人 · 暗光不眩光

夜起夜里 不被屏幕扎眼

老花眼对眩光特别敏感，纯黑底配纯白字反而最累。所以暗色背景 #0F1419 留一点墨蓝灰、文字 #F1F5F9 不用纯白；从近黑到深炭分两档，从象牙白到雾灰分两档，夜里起来看时间、看来电姓名都不晕。
