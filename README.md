# OldLauncher

OldLauncher 是一个面向老年用户的 Android 本地桌面应用，当前目标是把常用入口、联系人电话和基础设置收拢到一个更简单的启动器里。

## 当前项目范围

- 桌面主页与默认 Launcher 入口
- 应用管理与桌面应用选择
- 电话联系人查看、添加、编辑、删除、拨号
- 微信视频联系人页与实验性入口

## 当前不作为已交付能力

- 微信自动化拨号稳定能力
- 微信或系统电话自动接听
- 通知监听、远程协助、云同步
